# Sol De Cheiro

Landing page institucional e comercial para a marca Sol De Cheiro, especializada em sabonetes artesanais, produtos naturais e cuidado diário com a pele.

## Como rodar localmente

Instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Depois acesse:

```bash
http://localhost:3001
```

Para gerar a versão estática de publicação:

```bash
npm run build
```

## Onde alterar contatos e conteúdos

Os principais dados editáveis ficam em:

```bash
src/data/site.ts
```

Nesse arquivo é possível trocar o link do WhatsApp, redes sociais, produtos, benefícios, ingredientes, passos de compra, comparativo e FAQ.

Para um passo a passo mais simples, abra:

```bash
GUIA-DE-EDICAO.md
```
